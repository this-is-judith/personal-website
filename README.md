# Judith's Personal Website

Welcome! This is the repository for my personal website still under construction.

## Technologies Used

- HTML/CSS
- JavaScript
- React
- Node.js
- Firebase

## Features To Include

- **Multi-Page Layout**: Organization into multiple pages including Home, About, Experience, Projects, Blog, and Resources for easy navigation.
- **Mobile Responsive Design**: Crafting to provide an optimal viewing experience—easy reading and navigation with minimal resizing, panning, and scrolling—across a wide range of devices.
- **Dynamic Content**: Integration with CMS for easy updates and management of the blog and portfolio sections.
- **Contact Form**: Contact form for easy communication, making sure messages from visitors reach me directly.
